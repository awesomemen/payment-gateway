# 交付文档索引

本目录用于沉淀“面向正式用户验收”的交付材料，而不是仅供开发阶段使用的本地 smoke 文档。

当前包含：

- `user-acceptance-checklist.md`
- `environment-preparation-checklist.md`
- `configuration-inventory.md`
- `release-rollback-guide.md`
- `troubleshooting-guide.md`
- `evidence-package-template.md`
- `local-evidence-index.md`
- `local-reliability-suite.md`

使用顺序建议：

1. 先看 `environment-preparation-checklist.md`
2. 再看 `configuration-inventory.md`
3. 发布或重建环境时看 `release-rollback-guide.md`
4. 用户现场验收时使用 `user-acceptance-checklist.md`
5. 收集验收材料时使用 `evidence-package-template.md`
6. 回看当前仓库已沉淀的本地证据时看 `local-evidence-index.md`
7. 本地跑组件故障探针时看 `local-reliability-suite.md`
8. 出现异常时看 `troubleshooting-guide.md`
