Windows 常用开发命令：\n- 全仓测试：`.\mvnw.cmd --% -q -gs .mvn/global-settings.xml -s .mvn/settings.xml test -Dsurefire.failIfNoSpecifiedTests=false`\n- 指定模块测试：`.\mvnw.cmd --% -q -gs .mvn/global-settings.xml -s .mvn/settings.xml -pl gateway-application -am test -Dtest=SomeTest -Dsurefire.failIfNoSpecifiedTests=false`\n- 快速打包：`.\mvnw.cmd --% -q -gs .mvn/global-settings.xml -s .mvn/settings.xml -pl gateway-app -am package -Dmaven.test.skip=true`\n- 启动本地依赖和应用：`docker compose -f docker/local-compose.yml up -d --build gateway-app`\n- 查看运行态健康：`Invoke-RestMethod http://localhost:18081/actuator/health`\n- 运行本地可靠性套件：`& .\scripts\local-reliability-suite.ps1`\n- 查看仓库状态：`git status --short`\n说明：修改涉及 Docker 运行态时，需要实际重建 `gateway-app` 并验证健康状态，避免源码和容器漂移。
